
# sencha image grid list

## Outline

grid component to display some images by using sencha touch 2 

## Function

* Grid display

	The image display in a grid pattern.

* Selectable

	The images can be selected.

## Install via bower

```bash
  bower install sencha-image-grid-list --save
```
